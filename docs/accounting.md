# 会计

网络上已经有很多有关会计入门的内容, 可以搜索查看. 对于个人记账而言粗略了解即可. 

可以看如下日记账体会:

```
; 后付费
2021-01-01 * 燃气
    负债:应付账款:燃气      -10.00 CNY
    支出:燃气费             10.00 CNY

2021-02-01 * 燃气
    负债:应付账款:燃气      -10.00 CNY
    支出:燃气费             10.00 CNY

2021-03-01 * 燃气
    负债:应付账款:燃气      -10.00 CNY
    支出:燃气费             10.00 CNY
    
2021-03-02 * 付燃气费
    负债:应付账款:燃气       30.00 CNY
    资产:支付宝             -30.00 CNY
```

```
; 大件物品
2021-01-01 * 购买电脑
    资产:固定资产             4000.00 CNY
    资产:支付宝             -4000.00 CNY
    
; 如果1年后出售
2022-01-01 * 出售电脑
    资产:固定资产             -2500.00 CNY
    资产:支付宝                2500.00 CNY
    
; 每个月的电子产品支出为 1500 / 12 = 125
2022-01-01 * 电脑损耗
    资产:固定资产             -125.00 CNY
    支出:电子产品              125.00 CNY
```

可以看出对于非复式的记账软件, 同时管理消费和资产是做不到的. 甚至正确的管理消费都比较困难.
